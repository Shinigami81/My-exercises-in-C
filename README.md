🚀 My Exercises in C

Welcome to my personal repository where I document my journey learning the C programming language — from the basics to more advanced concepts.

📖 About This Repository

This repo is a collection of exercises designed to track my progress while learning C.

Rather than relying only on theory, I’m following a hands-on approach, because in my opinion that’s the most effective way to truly understand programming.

If you're learning C too, you might find this structure useful.

🧠 Learning Approach

I’m working through these exercises at my own pace no rush, no pressure.

To keep things structured and motivating, I’ve divided the journey into “seasons”, each containing a set of exercises aimed at building specific skills step by step.

📌 The Daily Path (The Method)

Instead of trying to learn everything at once, I follow a weekly routine similar to a “new hire” developer workflow:

Monday → Read and understand the task
Tuesday – Wednesday → Write code, struggle when it doesn’t compile, search for solutions (StackOverflow, MSDN, man pages)
Thursday → Debug using tools like gdb (Linux) or the Visual Studio debugger (breakpoints, variables, registers)
Friday → Refactor: clean up the code and add comments
Saturday → Rest or move to the next task
Sunday → Rest
💡 Final Thoughts

If you're learning C, I highly recommend combining theory (books) with a practical, hands-on approach like this one.

Consistency beats intensity. Keep coding. 👨‍💻🔥
